Opis projektu

Projekt polegał na stworzeniu od podstaw tej strony portfolio, a następnie uruchomieniu jej na lokalnym serwerze opartym o Raspberry Pi. Strona została zbudowana z użyciem czystego HTML, CSS i JavaScript, a do generowania części treści i stylów wykorzystano narzędzia AI. Serwer WWW został skonfigurowany z użyciem Node.js i Express.

Technologie

HTML5 CSS3 JavaScript Raspberry Pi Node.js Express
